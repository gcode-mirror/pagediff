Tool to monitor page changes, archive copies and alert users

Features
  * multiple users
  * send alerts by mail and rss feeds
  * monitor multiple pages
  * keep copies of the pages for _n_ days

Requirements
  * PHP5 (>5.2.4)
  * sqlite
  * web server
  * PEAR (TextDiff)
  * curl
  * Crontab (or maybe a web equivalent)
